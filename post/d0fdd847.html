Linux的基础命令

发表于2023-06-11|更新于2023-06-21|学习笔记
|阅读量:

Linux命令

cd 跳转文件夹

ls 显示目录

cd . 跳转为同级目录的其他目录

cd .. 跳转为上一级目录

cd ~ 跳转为home（root）目录

ls -a 显示隐藏目录

ls -l 竖着显示目录（更多内容）

ls -lah （h显示内存大小）

pwd 显示当前位置

mkdir 创建文件夹

mkidr -p 创建多级文件

touch 创建文件

ctrl+l 清空

cat 查看文件内容

more 多页查看内容（空格下一页 q退出）

cp （-r） 路径1 路径2 （复制文件夹要-r）

mv 路径1 路径2 （移动）

rm 删除 （文件夹加-r）

rm test* 删除前缀为test的东西

rm *test 删除后缀为test的东西

rm * test * 删除区间带test的东西

su - root 进入超级管理员 /sudo /root

exit 退出超级管理员（退出会话）

which 查找命令文件

find 起始路径 -name “文件名” （可同上的通配符*）

find / -size -(+) n(数字)[kmg] 按文件大小查找文件（-、+ 小于大于 ）

ctrl + c中断查询
